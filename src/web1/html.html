HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas web. Ela estrutura o conteúdo da web, permitindo a incorporação de texto, imagens, vídeos e outros elementos multimídia. O HTML utiliza uma série de tags e atributos para definir a aparência e o comportamento dos elementos na página, facilitando a navegação e a interatividade para os usuários.
